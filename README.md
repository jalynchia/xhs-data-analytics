# 背景

小红书创作服务平台的内容数据仅保存一段时间，但是从内容创作者角度出发，不仅需要在内容上线后的一段时间里观察和分析数据，还需要将数据留存更长时间，方便后续在多个内容之间进行对比、交叉分析等，来得出一些与内容创作、渠道运营有关的规律，从而指导后续的创作。

本 Skill 可对从小红书创作服务平台下载或导出的视频数据进行提取、合并和深度诊断，将数据明细合成到一张 Excel 表格，并生成带有交互图表的 HTML 仪表盘，帮助创作者更高效地进行数据分析与决策。

# 如何使用

## 数据准备

首先要登录[小红书创作服务平台](https://creator.xiaohongshu.com/new/home?source=official)。

本数据分析需要的数据清单如下：

  1. 内容条目的 `观看数据`，获取方式是：
    - 依此点击左侧边栏的「数据看板 > 内容分析」，进入「笔记数据列表」页面；
    - 在「笔记数据列表」找到需要分析的内容条目，点击最后一列的「详细数据」进入该条目的「笔记数据详情」页面。
    - 在「笔记数据详情」页面，点击「核心数据 > 基础数据」模块右侧的「导出数据」按钮 ✅
  2. 内容条目的 `互动数据`，获取方式是：
    - 在「笔记数据详情」页面，点击「核心数据 > 互动数据」模块右侧的「导出数据」按钮 ✅
  3. 内容条目的 `观看趋势`，小红书后台并未提供此数据的下方入口，需要我们通过 Inspect 获取，具体方法如下：
    - 在「笔记数据详情」页面单击鼠标右键，在弹出的菜单中点击 `Inspect` 打开 DevTools 开发者工具；
    - 在 DevTools 点击顶部的「Network」，然后刷新一下页面；
    - 待页面刷新完成后，在 network 列表找到 name 以 `trend?note_id=`开头的数据字段，然后单击，就会在右侧区域看到 `观看趋势` 的 json 格式数据；
    - 复制其中的 json 内容，粘贴到文本文档保存为 `[标题]-观看趋势.json` ✅
  4. 内容条目的 `观看来源` 、`观众画像`数据，直接在「笔记数据详情」页面完整截图即可。
  5. 如果有视频内容的字幕文件，也可以准备好一起拿来分析，可以帮助 LLM 更好地理解内容，并进行更有针对性的分析。

## 数据文件简单预处理

- 对上述 `基础数据`、`互动数据` 文件进行重命名，格式是：
    - 「[标题]-播放数据明细表.xlsx」
    - 「[标题]-互动数据明细表.xlsx」

## 开始分析

上述资料准备好放到一个文件夹里面，就可以直接在 Agent 中调用 skill 并说明数据源文件的文件夹路径来进行分析了。

> 生成的 HTML 仪表盘内嵌了外部 CDN 链接（用于加载 Google Fonts, Tailwind CSS, ECharts）。请确保您的查看终端处于有网络连接的环境下，否则图表与排版将无法正常渲染。
