用户每次提供以下原始数据 **3 个文件 + 2 张截图**（文件名含视频标题）：

| 输入类型            | 典型文件名模式               | 内容                                   |
| ------------------- | ---------------------------- | -------------------------------------- |
| `播放数据明细` xlsx | `[标题]-播放数据明细表.xlsx` | 播放时序数据 + 汇总指标（含粉丝对比）  |
| `互动数据明细` xlsx | `[标题]-互动数据明细表.xlsx` | 互动时序数据 + 汇总指标（含粉丝占比）  |
| `观看趋势` json     | `[标题]-观看趋势.json`       | 逐秒留存率：本视频 + 同类视频对比      |
| `观看来源` 数据图   | （无）                       | 各来源占比（首页推荐/视频推荐/搜索等） |
| `观众画像` 数据图   | （无）                       | 观众的性别/年龄/城市/兴趣分布          |

> **数据图提取**：观看来源和观众画像须通过视觉识别数据图读取数值。读取后直接以小数形式存储（如图中 `50.2%` 保存为 → `0.502`）。

# 1. 指标名称及含义勘误

`播放数据明细` xlsx 中「基础数据总览」sheet 的几个指标名称带有误导，实际含义以此处为准，并且后续使用 these 指标以及输出数据汇总时，需要使用更正后的指标名：

| 指标名                     | 实际含义               | 更正后的指标名          |
| -------------------------- | ---------------------- | ----------------------- |
| `封面点击率粉丝占比 (%)`   | 粉丝群体的封面点击率   | 粉丝 - 封面点击率(%)    |
| `平均观看时长粉丝占比 (%)` | 粉丝群体的平均观看时长 | 粉丝 - 平均观看时长 (s) |
| `完播率粉丝占比 (%)`       | 粉丝群体的完播率       | 粉丝 - 完播率 (%)       |
| `2秒退出率粉丝占比 (%) `   | 粉丝群体的2秒退出率    | 粉丝 - 2秒退出率 (%)    |


# 2. `播放数据明细` xlsx 的 sheet 结构说明

| Sheet 名               | 列结构              | 数值格式                              |
| ---------------------- | ------------------- | ------------------------------------- |
| `基础数据总览`         | 指标 / 数值         | 混合（整数、百分比、秒数）            |
| `曝光数（天）`         | 日期 / 曝光数       | 日期字符串 / 整数                     |
| `观看数（天）`         | 日期 / 观看数       | 日期字符串 / 整数                     |
| `封面点击率（天）`     | 日期 / 封面点击率   | 日期字符串 / 百分比字符串（如 `8%`）  |
| `平均观看时长（天）`   | 日期 / 人均观看时长 | 日期字符串 / 秒数字符串（如 `62s`）   |
| `平均观看时长（小时）` | 时刻 / 人均观看时长 | 时刻字符串 / 秒数字符串（如 `299s`）  |
| `完播率（天）`         | 日期 / 完播率       | 日期字符串 / 百分比字符串             |
| `完播率（小时）`       | 时刻 / 完播率       | 时刻字符串 / 百分比字符串             |
| `2s退出率（天）`       | 日期 / 2s退出率     | 日期字符串 / 百分比字符串             |
| `2s退出率（小时）`     | 时刻 / 2s退出率     | 时刻字符串 / 百分比字符串（可能为空） |
| `涨粉数（天）`         | 日期 / 涨粉数       | 日期字符串 / 整数                     |
| `涨粉数（小时）`       | 时刻 / 涨粉数       | 时刻字符串 / 整数                     |

# 4. `观看趋势` json 结构说明

```json
{
  "data": {
    "video": { ... },          // 不使用
    "no_data": false,
    "trend_list": [            // 本视频观看趋势
      {"date": 0, "count": 100},
      {"date": 1, "count": 93},
      ...
    ],
    "similar_trend_list": [    // 同类视频观看趋势
      {"date": 0, "count": 100},
      {"date": 1, "count": 81},
      ...
    ],
    "summary": "..."           // 文字说明，不使用
  }
}
```

字段映射：
- date  → 播放进度（秒）
- count → 留存率（0–1，单位为百分比）
